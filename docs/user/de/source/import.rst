Importieren
===========

Der GeoBox-Client bietet Ihnen die Möglichkeit, Raster- und Vektordaten zu importieren.
Wenn Sie eigene Shapedateien besitzen, die Sie im GeoBox-Client nutzen möchten, können Sie diese über den Menüpunkt `Shapedatei importieren` in den GeoBox-Client laden.
Sollten Sie Vektordaten im GeoJSON-Format nutzen, steht Ihnen für diesen Dateityp der Menüpunkt `GeoJSON-Datei importieren` zur Verfügung.
Um Rasterdaten lokal verfügbar zu haben, können Sie die über die GeoBox-Infrastruktur angebotenen Dienste über den Menüpunkt `Rasterdaten importieren` auf Ihrem PC speichern.
